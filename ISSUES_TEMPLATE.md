## Como Criar as Issues

### Opção 1: Via CLI (Recomendado)
```bash
# Primeiro, faça login
gh auth login

# Depois execute os comandos abaixo (um de cada vez)

# Issue 1
gh issue create --title "📚 FASE 1: Pesquisa & Clarificação de Escopo" --body "[copie o body acima]" --label "documentation,skills,fase-1"

# Issue 2
gh issue create --title "📝 FASE 2: Estruturação & Escrita da Documentação" --body "[copie o body acima]" --label "documentation,skills,fase-2"

# ... etc para todas as issues
```

### Opção 2: Manualmente no GitHub
1. Vá para: https://github.com/willian-silva01/Teste_Software_2/issues
2. Clique em "New issue"
3. Copie título e body de cada issue acima
4. Adicione labels
5. Clique em "Submit new issue"
